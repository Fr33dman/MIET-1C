# Лабораторная работа №8 — Полнотекстовый поиск

## Цель

Цель работы – изучить возможности платформы «1С:Предприятие 8.2» по настройке полнотекстового поиска и получить практические навыки работы с настройками индексирования.

## Среда выполнения

- **ОС**: macOS
- **Платформа 1С**: (8.3.27)
- **Тип ИБ**: файловая

## Порядок выполнения (чеклист)

- [ ] 1. Открыть конфигуратор и проверить, что для всех объектов и их реквизитов режим полнотекстового поиска установлен как «Использовать».
- [ ] 2. Открыть режим отладки. Перейти: «Все функции» → «Стандартные» → «Управление полнотекстовым поиском».
- [ ] 3. Включить полнотекстовый поиск (Настройка… → флаг включения), если выключен.
- [ ] 4. Обновить индекс; проверить дату актуальности индекса.
- [ ] 5. Создать отчёт `ПоискДанных`, отнести к `Общее`, создать основную форму.
- [ ] 6. Создать реквизит формы `ПоисковоеВыражение` (заголовок «Фраза»), добавить на форму.
- [ ] 7. Создать команду `Поиск`, назначить обработчик (пока пустой), вынести на форму; для кнопки включить «Кнопка по умолчанию».
- [ ] 8. Создать реквизит `СообщениеОРезультате`; элемент формы: «Поле надписи», заголовок — «Нет».
- [ ] 9. Создать реквизит `РезультатПоиска`; элемент формы: «Поле HTML документа», заголовок — «Нет».
- [ ] 10. Создать команды `ПредыдущаяПорция` и `СледующаяПорция`, объявить обработчики.
- [ ] 11. Создать реквизиты: `РезультатыПоиска` (СписокЗначений) и `ТекущаяПозиция` (Число) — можно не добавлять на форму.
- [ ] 12. Для созданных команд задать код обработчиков (по методичке).
- [ ] 13. Создать процедуру `Искать` (клиентская) — запуск серверной обработки + проверка ввода (по методичке).
- [ ] 14. Создать процедуру `ИскатьСервер` — получение результатов поиска (по методичке).
- [ ] 15. Создать обработчик нажатия для `РезультатПоиска` (по методичке).
- [ ] 16. Обновить конфигурацию БД, проверить поиск в режиме «1С:Предприятие».
- [ ] 17. Выгрузить `.dt` (если требуют).
