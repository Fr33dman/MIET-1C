# Лабораторная работа №6 — Отчеты. Макеты печати

## Цель

Цель работы – изучить возможности платформы «1С:Предприятие 8.2» по созданию отчетов и макетов для печати документов и получить практические навыки работы с данными объектами платформы.

## Среда выполнения

- **ОС**: macOS
- **Платформа 1С**: (8.3.27)
- **Тип ИБ**: файловая

## Порядок выполнения (чеклист)

- [ ] 1. Создать отчет `ОтчётПоМатериалам`, отнести к `Общее`.
- [ ] 2. Открыть схему компоновки данных (СКД) → «Готово».
- [ ] 3. Добавить набор данных `Остатки` (тип: запрос).
- [ ] 4. В конструкторе запроса собрать запрос как в методичке.
- [ ] 5. Проверить, что набор данных сформирован корректно (как на рисунке).
- [ ] 6. Переименовать путь поля `КоличестваОстаток` на `Количество`.
- [ ] 7. Вкладка «Настройки»: добавить группировку (по умолчанию).
- [ ] 8. В «Выбранных полях» указать: `Номенклатура`, `Склад`, `Количество`.
- [ ] 9. В «Ресурсах» указать ресурс: `Количество`.
- [ ] 10. В «ДетальныхЗаписях» указать `Номенклатура`; добавить группировку по `Склад`.
- [ ] 11. Скомпилировать и проверить отчёт.
- [ ] 12. Аналогично создать отчёт по прибыли от продаж (по методичке).
- [ ] 13. Открыть документ `ПродажаТоваров` → вкладка «Макеты» → конструктор печати.
- [ ] 14. В конструкторе печати: создать новую команду; распределить реквизиты (шапка/табличные части/подвал) как в методичке.
- [ ] 15. В макете: раздвинуть область `УслугиШапка`; назначить имя области `ТоварыВсего` после `Товары`.
- [ ] 16. Аналогично добавить область `УслугиВсего` после `Услуги`.
- [ ] 17. Обновить конфигурацию БД, проверить формирование отчётов и печать.
- [ ] 18. Выгрузить `.dt` (если требуют).
