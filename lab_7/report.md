# Лабораторная работа №7 — Пользователи. Права и роли

## Цель

Цель работы – изучить возможности платформы «1С:Предприятие 8.2» по настройке ролей пользователей и получить практические навыки работы с данными объектами платформы.

## Среда выполнения

- **ОС**: macOS
- **Платформа 1С**: (8.3.27)
- **Тип ИБ**: файловая

## Порядок выполнения (чеклист)

- [ ] 1. В режиме «1С:Предприятие» (Администрирование → Пользователи) создать 3 пользователей: `Администратор`, `МенеджерПоПродажам`, `МенеджерПоЗакупкам`.
- [ ] 2. Назначить им соответствующие права.
- [ ] 3. Создать справочник `Пользователи`, отнести к `Общее → Справочники`.
- [ ] 4. Запустить «1С:Предприятие», заполнить справочник `Пользователи` значениями (3 записи).
- [ ] 5. Создать константу `ТекПольз` (Строка 50).
- [ ] 6. В модуле сеанса конфигурации создать процедуру для заполнения `ТекПольз` при начале сеанса (как в методичке).
- [ ] 7. Создать форму списка для справочника `Пользователи`.
- [ ] 8. Для реквизита списка включить «Динамический список» и настроить произвольный запрос через конструктор (по методичке).
- [ ] 9. Для реквизита `Код` установить пользовательскую видимость только для полных прав.
- [ ] 10. Для ролей `ОтделПродаж` и `ОтделЗакупок` настроить права (с учётом связанных объектов).
- [ ] 11. Запустить режим отладки и проверить работоспособность.
- [ ] 12. Выгрузить `.dt` (если требуют).
